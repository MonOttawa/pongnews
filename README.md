# Pong News

Peer-reviewed research on table tennis health benefits. Built with Astro 6, Fraunces variable typography, zero JavaScript.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# Output: dist/
```

## Deployment

Production deploys to Cloudflare Pages from GitHub on every push to `main`.

- Build command: `npm run build`
- Output directory: `dist`
- Live: https://pongnews.pages.dev
